# Welcome to AutoElite Motors Assistant (Full Monitoring Edition)

The complete AI assistant with observability and evaluation.

## Features

- **Full RAG Pipeline**: Retrieval-augmented generation over all dealership documents
- **Context Engineering**: Structured prompts, conversation memory, smart retrieval
- **Observability**: Every query is traced with Arize Phoenix
- **Dual Mode**: Customer and Employee views

## Monitoring

Open the Phoenix dashboard at [localhost:6006](http://localhost:6006) to see:
- Query traces and latency
- Token usage per request
- Retrieval relevance scores
